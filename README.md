# AI Assistant Agent

## Overview
This project is a personal AI assistant built using Python and LLM-based agents. 
It is designed to answer questions, retrieve documents, and perform automated tasks.

## Features
- Conversational AI interface
- Long-term memory using vector databases
- Tool usage (file access, APIs)
- Modular agent architecture

## Tech Stack
- Python
- OpenAI API
- LangChain
- Chroma Vector DB
- Streamlit (UI)
